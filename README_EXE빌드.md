# EXE 파일 빌드 가이드

## 방법 1: 자동 빌드 스크립트 사용 (권장)

### Windows에서:

1. `build_exe.bat` 파일을 더블클릭하여 실행
   - 또는 명령 프롬프트에서 실행:
   ```cmd
   build_exe.bat
   ```

2. 빌드가 완료되면 `dist` 폴더에 `반편성배정프로그램.exe` 파일이 생성됩니다.

### 필요한 사항:

- Python이 설치되어 있어야 합니다 (개발자용)
- PyInstaller가 자동으로 설치됩니다

## 방법 2: 수동 빌드

### 1. PyInstaller 설치

```bash
pip install pyinstaller
```

### 2. EXE 파일 생성

```bash
pyinstaller --name=반편성배정프로그램 --onefile --windowed --add-data="출력서식.xlsx;." app.py
```

또는:

```bash
python setup_exe.py
```

### 3. 생성된 파일

- `dist/반편성배정프로그램.exe` - 실행 파일
- 이 파일을 다른 컴퓨터로 복사하여 실행 가능합니다.

## 주의사항

1. **출력서식.xlsx 포함**: 
   - exe 파일과 같은 폴더에 `출력서식.xlsx` 파일이 있어야 합니다.
   - 또는 exe 파일에 포함시켜야 합니다.

2. **첫 실행 시 방화벽 경고**:
   - Windows 방화벽에서 네트워크 접근 허용을 요청할 수 있습니다.
   - 허용해주세요 (로컬 서버 실행용).

3. **파일 크기**:
   - exe 파일은 약 100-200MB 정도 될 수 있습니다.
   - 모든 Python 라이브러리가 포함되기 때문입니다.

## 배포 방법

1. `dist/반편성배정프로그램.exe` 파일을 복사
2. `출력서식.xlsx` 파일도 함께 복사 (같은 폴더에)
3. 두 파일을 함께 배포

## 실행 방법

- `반편성배정프로그램.exe`를 더블클릭하면 자동으로 브라우저가 열립니다.
- 브라우저에서 프로그램을 사용할 수 있습니다.

## 문제 해결

### exe 파일이 실행되지 않는 경우:

1. Windows Defender나 백신 프로그램이 차단할 수 있습니다.
2. "신뢰할 수 있는 앱"으로 추가해주세요.

### 브라우저가 열리지 않는 경우:

1. 수동으로 브라우저를 열고 `http://localhost:8501`로 접속하세요.
