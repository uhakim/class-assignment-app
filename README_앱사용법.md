# 반편성 배정 프로그램 사용법

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 프로그램 실행:
```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 프로그램이 실행됩니다.

## 사용 순서

### 1단계: 학생자료 업로드
- `학생자료.xlsx` 파일을 업로드합니다.
- 파일 형식은 기존과 동일해야 합니다.

### 2단계: 분리명부 업로드
- `separation.xlsx` 파일을 업로드합니다.
- 분리해야 하는 학생 쌍 정보가 포함되어야 합니다.

### 3단계: 배정표 다운로드
- 자동으로 `반편성_배정표.xlsx` 파일이 생성됩니다.
- 다운로드하여 수동 배정을 완료합니다.

### 4단계: 완료 파일 업로드
- 수동 배정을 완료한 `반편성_완료.xlsx` 파일을 업로드합니다.

### 5단계: 최종 출력서식 생성
- `출력서식_완료.xlsx` 파일이 생성됩니다.
- 다운로드하여 사용합니다.

## 주의사항

- `출력서식.xlsx` 템플릿 파일이 현재 디렉토리에 있어야 합니다.
- 각 단계를 순서대로 진행해야 합니다.
- 파일 업로드 시 기존 파일명과 동일한 이름을 사용하는 것이 좋습니다.
